# To-do API


## ROUTES TO IMPLEMENT
| METHOD | ROUTE | FUNCTIONALITY |
| ------- | ----- | ------------- |
| *GET* | ```/todos/``` | _Get all todos_|
| *POST* | ```/todos/``` | _Create a todo_|
| *GET* | ```/todo/{id}``` | _Get a todo by id_|
| *PUT* | ```/todo/{id}/``` | _Update a todo_|
| *PATCH* | ```/todo/{id}/``` | _Update todo status_|
| *DELETE* | ```/todo/{id}/``` | _Delete a todo_|
